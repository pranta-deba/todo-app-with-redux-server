# Todo Task Management App

- Fetch all todos with optional priority filtering
- Add a new todo
- Update an existing todo
- Delete a todo
- Efficient state management using Redux Toolkit Query

## Tech Stack

### Frontend:

- React.js
- Redux Toolkit Query
- TypeScript
- Tailwind CSS

### Backend:

- Node.js
- Express.js
- MongoDB
